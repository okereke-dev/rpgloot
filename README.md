# RPGLoot

A Paper 26.1.2 plugin that adds an RPG rarity system to weapons. Hostile mobs drop weapons with random names, scaled stats, and bonus effects — no crafting required.

**5 tiers:** Common · Uncommon · Rare · Hero · Legendary

## Installation

1. Place `rpgloot.jar` in your server's `plugins/` folder
2. Restart the server

**Requires:** Paper 26.1.2 · Java 21
